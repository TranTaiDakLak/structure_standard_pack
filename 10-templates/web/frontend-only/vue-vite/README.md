# Web Frontend — Vue + Vite

## Khi nào dùng

- Admin portal, dashboard, web app nhỏ-vừa

## Có gì trong thư mục này

- `simple.md`: dùng khi cần ship nhanh, repo còn gọn

## Ghi chú

- Vue/Vite thường là frontend web mặc định gọn, nhanh, dễ mở rộng.
