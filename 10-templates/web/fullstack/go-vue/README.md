# Web Fullstack — Go + Vue

## Khi nào dùng

- 1 frontend + 1 backend trong cùng product
- simple cho team nhỏ, structured khi cả hai phía đều bắt đầu phình

## Có gì trong thư mục này

- `simple.md`: dùng khi cần ship nhanh, repo còn gọn
- `structured.md`: dùng khi code bắt đầu phình và cần boundary rõ hơn

## Ghi chú

- `backend/` + `frontend/` hợp lệ với fullstack nhỏ-vừa.
